🔗 PeerJS (WebRTC P2P) 🗺 On roadmap

Truly peer-to-peer once connected — no relay server carries your data. Low latency. Uses PeerJS cloud for the initial handshake only. Best for privacy since data flows directly browser-to-browser.

✓ No relay for data   ✓ Very low latency   ✓ Free signaling tier   ✓ Private
✗ NAT traversal fails ~10% of connections   ✗ More complex setup   ✗ Needs signaling key
