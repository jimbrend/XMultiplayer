Full dashboard requires the local server. Running node server.js imports your seen history from this extension into the 𝕏 Multiplayer Dashboard and unlocks multiplayer, Raindrop, and the onboarding tutorial.
cd x-multiplayer && node server.js
Verify source on GitHub ↗

After the server is running, open http://localhost:3000 — the multiplayer tutorial starts automatically on first launch.

Or open index.html directly — launches the dashboard in a new tab (limited without the server; history sync works best with node server.js).
